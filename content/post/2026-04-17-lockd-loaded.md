This week has been all about LocalLLM experimentation.

- [Google Releases Gemma4][1] - Google DeepMind released Gemma4 and I've been
  experimenting with it. The boilerplate for this post's markdown file was
  automatically written by gemma4:26b. It wasn't as fast as Claude Code, but it
  worked equally well.

- [Jan.ai][2] - I found out about this in an HN thread suggesting alternatives
  to [Ollama][3]. I'm a little sketched out by how difficult it is to find info
  on who produces it - much lik LM Studio. But, it's open source and on Github,
  properly attributes the projects it's built on, and offeres a lot of
  customization, so I'm giving it a go.

- [LlamaBarn][4] - LlamaBarn is a macOS-native app to run LocalLLMs. I like
  where the project is heading, but there's a lot of functionality it doesn't
  have yet, so I'll keep checking back in on it as it improves.

That's all for this week! God bless.
